# Commentary Canvas

Commentaries are the working canvas for enhancing and leveraging ph-civ one chapter at a time. A seeded commentary is not finished analysis. It is a protected surface for later synthesis.

Every chapter commentary should preserve:

- the lecture transcript as the source body, without rewriting it
- existing commentary and source-backed notes
- open canvas metadata in frontmatter
- a `Project Canvas` section for future deepening

Required frontmatter:

- `commentary_status`
- `canvas_status: open`
- `analysis_depth: seed`
- `scaffold_version: ph_civ_commentary_canvas_v1`

Required canvas sections:

- `Project Leverage`
- `Laws / Patterns Exposed`
- `Volume Role`
- `Museum Hooks`
- `Strategy / Present-Day Application`
- `Counter-Readings`
- `Open Questions`
- `Build Notes / Future Enhancements`

The scaffold is intentionally permissive. It should invite chapter-by-chapter development without implying that the commentary is complete.
