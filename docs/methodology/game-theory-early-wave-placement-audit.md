# Game Theory Early-Wave Placement Audit

This audit covers gt-01 through gt-15 before any direct namespace migration.

The governing rule is conceptual fit rather than series unity. A chapter should land in ph-civ when it primarily discovers laws, structures, or civilizational formation; it should land in ph-apo when it primarily applies those laws to live crisis, escalation, or world-war conditions.

| Source ID | Title | Proposed Volume | Rationale |
| --- | --- | --- | --- |
| gt-01 | Game Theory #1: The Dating Game | ph-civ | Introduces game theory as a general model of human behavior and social prediction rather than applying it to a live geopolitical crisis. |
| gt-02 | Game Theory #2: Why Schools Suck | ph-civ | Uses schooling, class reproduction, and institutional incentives as law-discovery material about how societies shape behavior. |
| gt-03 | Game Theory #3: Rich Dad, Poor Dad | ph-civ | Focuses on hierarchy, class, parenting, trust, and social reproduction as structural rules rather than apocalypse-facing events. |
| gt-04 | Game Theory #4: The Immigration Trap | ph-civ | Uses immigration, status, and assimilation as a study of social game structure inside empire rather than a late-war application packet. |
| gt-05 | Game Theory #5: The World Game | ph-civ | Develops civilizational law around rise and fall, asabiyyah, energy, openness, and cohesion. |
| gt-06 | Game Theory #6: The World's Bank | ph-civ | Reconstructs the British imperial game and the monetary order as foundational system architecture. |
| gt-07 | Game Theory #7: America's Game | ph-civ | Explains the American nation-game, reserve-currency logic, and empire-building grammar as law-discovery material. |

## Proposed Split

- ph-civ: gt-01 through gt-08
- ph-apo: gt-09 through gt-15

## Notes

- This audit does not migrate gt-01 through gt-15; it only fixes conceptual destination before any next phase.
- The current legacy metadata still treats the whole early series as world-war, but this audit recommends a cleaner two-volume split.
- If the series is migrated later, use the same repo-local sources preservation rule so the full transcript body remains inside ph-civ.
