# Book

This folder is the canonical public reader root for the two-volume ph-civ artifact.

The staged redesign teaches a civilizational historiography directly in the filesystem:

- [Table of Contents](./table-of-contents.md) lists every current chapter and title across both volumes.
- [Volume I: Civilization](./volume-i-civilization/README.md) discovers civilizational laws.
- [From Civilization To Apocalypse](./parts/civilization-to-apocalypse.md) explains the hinge.
- [Volume II: Apocalypse](./volume-ii-apocalypse/README.md) applies those laws under pressure.
- [Provenance](./provenance/README.md) names the older source-family scaffold as subordinate archival history.

## Reading Contract

- civilizations are the primary historical unit
- Great Books function as civilizational evidence inside Volume I
- Secret History is mostly Apocalypse, with selective Civilization support only
- geo, gt, sh, and sub remain distinct Apocalypse spines but are all subordinate to application
- **lecture packets** (transcript + commentary) are canonical under [`lectures/`](../lectures/README.md); legacy `book/volume-*` paths are redirect stubs after PH-LECTURES

## Source-Lattice Rule

Each canonical chapter wrapper is a source-lattice doorway:

1. doorway in the staged canonical README
2. primary source floor in the existing legacy packet
3. commentary/orientation/card as bounded secondary support
4. widened interpretation only after the source floor is stable

Where meaningful, chapter doorways now also link outward into the larger CIV-STATE source-lattice through shelf-readers plus era-matched primary and secondary sources.
