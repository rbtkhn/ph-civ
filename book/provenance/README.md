# Provenance

This folder names the old source-family scaffold as provenance. In the staged pass, the old `book/volume-*` folders still carry the underlying packets, but they no longer define the intended public reading order.

- old Volume I: `book/volume-i/` for Geo-Strategy residue
- old Volume II: `book/volume-ii/` for Civilization residue
- old Volume III: `book/volume-iii/` for Game Theory residue
- old Volume V: `book/volume-v/` for Great Books residue
- old Volume VI: `book/volume-vi/` for Secret History residue
- old Volume VII: `book/volume-vii/` for Essays residue
