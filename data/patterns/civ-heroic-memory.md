# Heroic Memory

## Summary

A culture uses heroic narrative to preserve ideals, wounds, hierarchy, courage, grief, and moral imagination across generations.

## Strategy Use

Use when a live political or strategic actor relies on inherited memory, martyrdom, honor, humiliation, or epic self-description to organize action.

## Source Cards

- `civ-07`
- `gb-02`

## Limits

Do not reduce every memory system to Homer. Use the pattern only as a prompt to ask how narrative inheritance shapes conduct.
