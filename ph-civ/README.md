# ph-civ

`ph-civ` is the public namespace for Volume I / Predictive History: Civilization.

This surface discovers the laws of history through the Civilization-facing chapter layer. It contains lecture transcripts, companion commentaries, the active public card export, study prompts, literary spines, and validation flow for Volume I.

The repository is also named `ph-civ`; the repo as a whole is the two-volume ph-civ artifact containing `ph-civ`, `ph-apo`, and `ph-mus`.

The Python package still imports as `civ_ph` internally for compatibility, but the public command is `ph-civ`.
