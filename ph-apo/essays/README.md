# ph-apo Essays

This lane holds the Predictive History essay corpus that belongs to `ph-apo`.

These essays are written application texts rather than lecture transcripts. They still follow the same public discipline as the rest of the repository:

- preserve source fidelity in the transcript file
- place guardrails and limits in the commentary file
- expose a compact public orientation card in `data/cards/`

Use `sub-*` as the canonical public essay numbering namespace.

Volume VII remains provenance metadata for the physical source folders under `book/`. It is not the reader-facing routing rule.
